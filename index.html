Flexbox 101

To use flexbox we need to set the display property of an element as follows:

display: flex;

flex-direction

Specifies the direction in which the items inside the flexbox container will stack.

  • row: Stacks items horizontally (default).
  • row-reverse: Same as row but items go in reverse order.
  • column: Stacks items vertically.
  • column-reverse: Same as column but items go in reverse order.

flex-direction: row;

flex-wrap

Specifies if items would wrap according to screen size (resize viewport to notice).

  • nowrap: Items will not wrap (default).
  • wrap: Items will wrap if necessary.
  • wrap-reverse: Same as wrap but but items go in reverse order.

flex-wrap: nowrap;

flex-flow

Is a combination of flex-direction and flex-wrap properties into a single one

flex-flow: [direction] [wrap];

justify-content

Aligns flex items in main-axis (horizontally).

  • flex-start: Flex items will be at the start of the container (default)
  • flex-end: Flex items will be at the end of the container.
  • space-between: Flex items will have equal space between them.
  • space-around: Flex items will have a space before, between, and after them.
  • space-evenly: Flex items will space around evenly.
  • center: Flex items will go to the middle of the container

justify-content: flex-start;

align-items

Aligns flex items in cross-axis (vertically).

  • flex-start: Flex items will be at the top of the container (default)
  • flex-end: Flex items will be at the bottom of the container.
  • stretch : Flex items will stretch to fill remaining space (default).
  • center: Flex items will go to the middle of the container
  • baseline: Flex items will be aligned to their baseline

align-items: stretch;

align-content

Aligns wrapping flex items in cross-axis (vertically for flex-direction: row, horizontally for flex-direction: column).

Only has effect when the flex container has multiple lines of flex items.

  • flex-start: Flex items will be at the start of the container
  • flex-end: Flex items will be at the end of the container.
  • space-between: Flex items will have equal space between them.
  • space-around: Flex items will have a space before, between, and after them.
  • space-evenly: Flex items will space around evenly.
  • stretch : Flex items will stretch to take up remaining space (default).
  • center: Flex items will go to the middle of the container

align-content: stretch;
